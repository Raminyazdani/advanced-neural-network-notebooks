# Advanced Neural Network Regularization Techniques

Practical implementations of regularization methods for neural networks.

## Overview

This project demonstrates key regularization techniques for preventing overfitting in neural networks, including parameter norm penalties (L1, L2, Elastic Net) and modern deep learning methods (dropout, weight decay, early stopping).

## Tech Stack

- Python 3.8+
- PyTorch
- NumPy
- Matplotlib
- Jupyter Notebook
